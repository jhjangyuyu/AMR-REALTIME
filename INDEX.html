  1. Firebase 콘솔 (console.firebase.google.com) 접속 → 새 프로젝트 생성
  2. 프로젝트 → 빌드 → Realtime Database → 데이터베이스 만들기 (테스트 모드 선택)
  3. 프로젝트 설정 → 내 앱 → </> 웹 앱 추가 → firebaseConfig 객체 복사
  4. 아래 텍스트박스에 복사한 내용을 붙여넣고 저장
